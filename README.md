### Agent Roles

| Agent | Role |
|-------|------|
| **READER** | Reads PDF, extracts visualizable concepts |
| **VIZ_DESIGNER** | Designs how each concept should be visualized |
| **CODER** | Writes HTML/CSS/JS (D3.js) code |
| **REVIEWER** | Checks for bugs, errors |
| **CRITIC** | Points out flaws, suggests improvements |
| **RATER** | Scores result 1-10, decides if iteration needed |
| **ORCHESTRATOR** | Manages the loop, triggers right agent for fixes |

### OpenClaw Commands Used

```bash
# Spawn agents for each role
sessions_spawn --runtime subagent --task "Read PDF and extract concepts"

# Agents communicate via sessions
sessions_send --sessionKey <agent> --message <concept_data>
```

1. Upload a math PDF to OpenClaw
